Luqenov Fedora Repository
=========================
A third-party RPM binary package repository for Fedora.

Setup:
  sudo dnf copr enable luqenov/bun
  sudo dnf copr enable luqenov/equibop-dev
  sudo dnf copr enable luqenov/electron35

Install a package:
  sudo dnf install bun
  sudo dnf install equibop-dev
  sudo dnf install electron35

Updates come through normally with:
  sudo dnf upgrade
